进程生命周期

从 Shell 按下回车到子进程退出——整个过程在操作系统中到底发生了什么？

1/8 初始状态
// Shell 执行命令
$ git status
// 1. Shell 调用 fork()
pid_t pid = fork();
// 2. 子进程调用 exec()
execvp("git", argv);
// 3. 父进程等待
waitpid(pid, &status, 0);
// 4. 退出码
echo $? # 0 = 成功
